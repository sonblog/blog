Fibonacci là gì ? Thôi google đi cho nhanh

f(n) = f(n-1) + f(n-2)



 	
  * n = 0, f = 0

 	
  * n = 1, f = 1

 	
  * n = 2, f = 1 (f(0) + f(1) = 0 + 1)


Dãy số fibonacci: 0, 1, 1, 2, 3, 5, 8, **13.** Con số 13 = 8 + 5 (tổng 2 số trước)

0: 0

1: 1

2: 1

3: 2

4: 3

5: 5

6: 8

7: 13
